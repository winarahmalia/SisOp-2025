### 2. Konversikan bilangan desimal di bawah ini ke dalam bilangan biner
a. *1234~10~*  
b. *5670~10~*  
c. *2321~10~*

*Jawaban:*  
a. *1234~10~* = *10011010010~2~*  
b. *5670~10~* = *1011000100110~2~*  
c. *2321~10~* = *100100010001~2~*

*Cara:*  
a. *1234~10~*  
1234 ÷ 2 = 617 sisa 0  
617 ÷ 2 = 308 sisa 1  
308 ÷ 2 = 154 sisa 0  
154 ÷ 2 = 77 sisa 0  
77 ÷ 2 = 38 sisa 1  
38 ÷ 2 = 19 sisa 0  
19 ÷ 2 = 9 sisa 1  
9 ÷ 2 = 4 sisa 1  
4 ÷ 2 = 2 sisa 0  
2 ÷ 2 = 1 sisa 0  
1 ÷ 2 = 0 sisa 1  

Hasil: *10011010010~2~*

### 11. Konversikan bilangan di bawah ini ke dalam bilangan desimal
a. *010100011,001111101~2~*

c. *4C5,2B8~16~*

*Jawaban:*

1 × 2^5 = 32

Bagian pecahan:  
0 × 2^-1 = 0  
0 × 2^-2 = 0  
1 × 2^-3 = 0,125

1 × 2^-6 = 0,015625  
1 × 2^-7 = 0,0078125

1 × 2^-9 = 0,001953125  

Total: 0,125 + 0,0625 + 0,03125 + 0,015625 + 0,0078125 + 0,001953125 = *0,245*

Hasil: *163,245~10~*
